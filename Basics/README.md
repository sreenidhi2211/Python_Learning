# Basics

## Topics Covered

### 1. Introduction To Python
**Status:** ✅ Complete
* **Hello World:** The basic structure of a Python script using the `print()` function.
* **Syntax:** How Python uses indentation to define code blocks instead of the curly braces used in C, C++, and Java.

******

### 2.  Keywords,Variables and Data Types
**Status:** ✅ Complete
* **Variables:** Understanding dynamic typing and Python's object-reference model, where variables act as references to objects.
* **Keywords:** Reserved words that define the structure of the code and cannot be used as identifiers.
* **Data Types:** Core types including numeric (`int`, `float`, `complex`), sequences (`str`, `list`, `tuple`), mapping (`dict`), sets, and booleans (`True`/`False`).

******

### 3. Input, Output Fromatting and Operators
**Status:** ✅ Complete
* **I/O:** Interacting with users via `input()` and displaying output with `print()`.
* **Formatting:** Formatting text using the modulo operator (`%`), the `.format()` method, and modern, highly efficient `f-strings`.
* **Operators:** Arithmetic, assignment, comparison, logical, bitwise, identity (`is`), and membership (`in`) operators. 
* **Evaluation:** The strict precedence hierarchy Python uses to evaluate expressions, and how associativity breaks ties.
* **Type Casting:**Type casting is converting a variable from one data type to another

******

### 4. Conditional Statements and Loops
**Status:** ✅ Complete
* **Branching Logic:** Using `if`, `elif`, and `else` to control the flow of execution based on specific conditions.
* **Pattern Matching:** Using the `match-case` statement to cleanly evaluate multiple exact patterns.
* **Iteration:** Using `for` loops to iterate over sequences and `while` loops to execute code based on a condition.
* **Loop Control:** Managing loop flow using `break` to exit, `continue` to skip, and the unique `else` clause to run code only if a loop finishes without breaking.

******

### 5. Lists
**Status:** ⏳ Pending
* **Core Properties:** Creating ordered, mutable collections that can store mixed data types and allow duplicate values.
* **Accessing & Slicing:** Retrieving elements using positive or negative indexing, and extracting specific ranges using the slicing operator..
* **Modifying Elements:** Updating values directly by index, and adding new elements using `append()`, `insert()`, and `extend()`.
* **Removing Items:** Deleting elements by value or index using built-in methods like `remove()`, `pop()`, `del`, and `clear()`.
* **Advanced Operations:** Iterating through list elements using `for` loops and locating specific items with the `index()` method.
